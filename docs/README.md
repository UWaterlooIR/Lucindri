# docs/

Reference material and design notes related to Lucindri.

## Contents

- **[indri-query-language.md](indri-query-language.md)** — a guide to the full Indri structured
  query language, annotated with what Lucindri actually supports (✅/⚠️/❌). Authored from
  knowledge of the language and a reading of Lucindri's parser; the authoritative upstream
  source is the Lemur Project wiki (linked inside the file).

- **[query-biased-summaries.md](query-biased-summaries.md)** — how to produce query-biased
  extractive summaries of a result document "the Lucene way" (`UnifiedHighlighter` with a sentence
  `BreakIterator`), since Lucindri lacks Indri's passage/extent operators; plus LM-faithful
  alternatives (ephemeral per-doc index; sentence-granular index).
- **[lucindri-vs-indri.md](lucindri-vs-indri.md)** — Lucindri↔C++ Indri query-language conformance
  catalog (generated by `../scripts/diff_indri.sh`).
